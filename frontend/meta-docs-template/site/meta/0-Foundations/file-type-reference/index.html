    Centralization Principle¶

    If a principle or definition repeats across docs, centralize it in Foundations or Operations.

    • Foundations - Core principles, standards, and definitions that apply universally
    • Operations - Process definitions, automation rules, and behavioral protocols
    • Avoid duplication - Reference centralized definitions rather than repeating them
    • Single source of truth - Each concept should have one authoritative location
